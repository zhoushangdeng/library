# 图书管理系统------

## 运行项目

- 安装依赖 `npm install`
- 启动项目 `npm run dev`
- 运行测试 `npm run test`

.env 配置样例：
```
#NODE_ENV=production
SERVER_PORT = 6006
NM_PUBLIC_KEY = "jwtSecret"
TOKEEN_EXPRIES = 684000

MYSQL_HOST = localhost
MYSQL_USER = root
MYSQL_PASS = root
MYSQL_DATABASE = librarian
MYSQL_PORT = 3306

SMTP_HOST = localhost
SMTP_PORT = 465
MAILBOX = test@qq.com
MAILBOX_PASS = qwertyuioasf
```
